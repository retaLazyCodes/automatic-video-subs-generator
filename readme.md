# Instrucciones para ejecutar


1. Instalar Python 3x y PIP
2. Instalar librería moviepy
~~~
    pip install moviepy
~~~
3. En el proyecto, editar el archivo 'set_path.py' y escribir la ruta de su directorio en el valor de la variable 'root_directory'
4. A traves de la terminal, situarse en el directorio donde está el proyecto
5. Finalmente, ejecutarlo con:
~~~
    py main.py   
~~~
